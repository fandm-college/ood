# File manager

OOD file manager feature similar to what you would find in Wndows/Mac OS X.
It  can be accessed by selecting:

- **Files**
- **Home Directory**

## Upload/download files

![OOD file manager menu item](../assets/images/file_manager.png)

{: .note }
